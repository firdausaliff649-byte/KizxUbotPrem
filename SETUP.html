🤖 KizxUbot Premium - Setup Assistant

Complete guide untuk setup bot di panel hosting Anda

📥 Step 1: Download Files

Option A: Download dari GitHub (Recommended)

1. Buka: https://github.com/firdausaliff649-byte/KizxUbotPrem

2. Click tombol hijau "Code"

3. Pilih "Download ZIP"

4. File: KizxUbotPrem-main.zip

Option B: Clone dengan Git
git clone https://github.com/firdausaliff649-byte/KizxUbotPrem.git

📤 Step 2: Upload ke Panel Hosting

Via cPanel File Manager:

1. Login ke cPanel

2. Buka "File Manager"

3. Masuk folder: public_html/

4. Click "Upload"

5. Upload file: KizxUbotPrem-main.zip

⚠️ Jika folder tidak ada, buat folder baru bernama "bot" atau "telegram-bot"

📦 Step 3: Extract Files

Menggunakan cPanel File Manager:

1. Cari file: KizxUbotPrem-main.zip

2. Right-click → Extract

3. Tunggu proses selesai ✅

4. Folder akan menjadi: KizxUbotPrem-main/

✅ Hasil extract: semua files sudah ada!

⚙️ Step 4: Konfigurasi .env

Edit file .env di panel:

1. Buka File Manager

2. Cari file: .env.example

3. Right-click → Edit atau Copy

4. Rename copy menjadi: .env

5. Edit isi file:

# Edit values ini: BOT_TOKEN=8796670391:AAESeHo9zhwB6RU4ebqik-MBZTjgNLvyU-4 ADMIN_IDS=123456789,987654321 DB_TYPE=sqlite BROADCAST_LIMIT=30
⚠️ PENTING: Ganti BOT_TOKEN dengan token Telegram Anda!

🔧 Step 5: Install Dependencies

Option A: Gunakan Terminal (SSH)
cd ~/public_html/KizxUbotPrem-main pip3 install -r requirements.txt
Option B: Via cPanel Terminal

1. Login cPanel

2. Cari "Terminal" atau "SSH"

3. Jalankan command di atas ☝️

ℹ️ Proses ini butuh 2-5 menit, tunggu sampai selesai!

🚀 Step 6: Jalankan Bot

Quick Test:
cd ~/public_html/KizxUbotPrem-main python3 main.py

Jika OK, tekan Ctrl+C untuk stop

Setup Auto-Start (Permanent):

Gunakan Supervisor agar bot jalan otomatis:

# Create config sudo nano /etc/supervisor/conf.d/kizx-bot.conf # Paste ini: [program:kizx-bot] command=/usr/bin/python3 /home/username/public_html/KizxUbotPrem-main/main.py directory=/home/username/public_html/KizxUbotPrem-main user=username autostart=true autorestart=true stderr_logfile=/home/username/logs/bot.err stdout_logfile=/home/username/logs/bot.log # Save: Ctrl+X → Y → Enter # Activate: sudo supervisorctl reread sudo supervisorctl update sudo supervisorctl start kizx-bot
✅ Bot akan jalan otomatis dan restart jika error!

🐛 Troubleshooting

❌ Bot tidak jalan?
# Check logs: tail -f ~/logs/bot.log # Verify .env: cat .env | grep BOT_TOKEN # Test Python: python3 --version
❌ Import Error?
# Reinstall packages: pip3 install --upgrade -r requirements.txt
❌ Permission Denied?
# Fix permissions: chmod 755 main.py chmod 755 start.sh
